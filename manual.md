## 必要な環境

| 項目 | 内容 |
|------|------|
| ブラウザ | Google Chrome、Microsoft Edge など（最新版を推奨） |
| 接続 | インターネット不要（ローカルで動作） |
| 入力 | キーボード。複数人プレイ時はゲームパッドも利用可能 |

**注意:** `index.html` をエクスプローラーからダブルクリックして開く（`file://` で開く）方法では、**JavaScript が読み込めずボタンが反応しない**ことがあります。必ず下記の「ローカルサーバー」経由で起動してください。

## ゲームの起動方法

### 推奨: Node.js の `serve` を使う

1. ターミナル（PowerShell など）を開く。
2. ゲームのフォルダに移動する。

   ```powershell
   cd "d:\work\game\tetris battle"
   ```

   ※ 自分の PC 上の実際のパスに読み替えてください。

3. 次のコマンドを実行する。

   ```powershell
   npx --yes serve .
   ```

4. 表示された URL（例: `http://localhost:3000`）をブラウザで開く。
5. **`http://` で開く**（`https://` ではない）。アドレスバーに `https` と出る場合は、先頭を `http` に直してください。

### その他の起動方法

| 方法 | コマンド例 | 備考 |
|------|------------|------|
| http-server | `npx --yes http-server -p 8080` → `http://localhost:8080` | MIME タイプが正しく返る |
| VS Code / Cursor Live Server | `index.html` を右クリック → Open with Live Server | 拡張機能が必要 |
| Python（標準） | `python -m http.server 8080` | **Windows では `.js` の MIME が誤り、動かないことがある**（後述） |

起動に成功すると、タイトル画面に「テトリスバトル」とモード選択ボタンが表示されます。

## 画面の流れ

```
タイトル画面 → ゲーム画面 → （制限時間到達時: TIME UP 表示）→ 結果画面 → （タイトルへ戻る）→ タイトル画面
```

| 画面 | 内容 |
|------|------|
| **タイトル画面** | 人数・モードを選ぶ |
| **ゲーム画面** | テトリスをプレイする |
| **TIME UP 表示** | 制限時間に達したとき、ゲーム画面中央に「TIME UP」と表示（約3秒間）。この間は操作できない |
| **結果画面** | 順位・プレイ時間・消したライン数などを表示 |

## モードの選び方

### メインモード（人数）

タイトル画面で次のいずれかを選びます。

| ボタン | 内容 |
|--------|------|
| **1人用** | 1人でプレイ（すぐにゲーム開始） |
| **2人対戦** | 2人で同時プレイ |
| **3人対戦** | 3人で同時プレイ |
| **4人対戦** | 4人で同時プレイ |

### サブモード（2〜4人のみ）

2人・3人・4人を選んだあと、**同じタイトル画面内**でサブモードを選びます。

| ボタン | 内容 |
|--------|------|
| **タイムアタック** | 制限時間 **5分** のうちにスコアを競う |
| **対戦モード** | 制限時間 **10分**。ライン消去で相手にお邪魔ブロックを送る |

- **1人用**は、常に **タイムアタックのみ** です（サブモード選択はありません）。
- サブモード選択中に **戻る** を押すと、メインモード選択に戻れます。

## 操作方法

### キーボード（人数ごとに割り当て）

同じ PC で複数人プレイする場合、プレイヤーごとに別のキーを使います。

| プレイヤー | 左 | 右 | 高速落下 | 回転 |
|------------|-----|-----|----------|------|
| **プレイヤー 1** | ← | → | ↓ | スペース |
| **プレイヤー 2** | A | D | S | Q |
| **プレイヤー 3** | J | L | K | U |
| **プレイヤー 4** | F | H | G | R |

#### 操作のポイント

- **左右:** 1マス移動。キーを**押し続けると連続**して移動します。
- **下:** ブロックの落下を速くします（ソフトドロップ）。
- **回転:** ブロックを時計回りに90度回転します。
- **ホールド**（キープ）機能はありません。
- **ハードドロップ**（一気に最下段まで落とす）機能はありません。

### ゲームパッド

2〜4人プレイでは **ゲームパッド** の利用を想定しています。

| 操作 | ゲームパッド |
|------|----------------|
| 左右移動 | 左スティック左右 / 十字キー左右 |
| 高速落下 | 左スティック下 / 十字キー下 |
| 回転 | A ボタン（ボタン0） |

- 接続したパッドは、**プレイヤー番号と同じ番号のコントローラー**（1P → 0番、2P → 1番 …）として割り当てられます。
- パッドを接続したあと、ブラウザ上でボタンを一度押すと認識されやすくなります。

## ゲーム画面の見方

各プレイヤーのパネルには次の情報があります。

| 表示 | 説明 |
|------|------|
| **メインの盤面** | 横12マス × 縦24マスのフィールド |
| **NEXT** | 次に落ちてくるブロック（1つ） |
| **SCORE** | タイムアタック時の得点（対戦モードでは非表示） |
| **LINES** | 消したラインの累計 |
| **半透明のブロック** | 落下予測位置（ゴースト）。現在のブロックが着地するおおよその位置 |

画面上部には、選択したモード名と、**制限時間があるモード**（タイムアタック・対戦）では**残り時間**が表示されます（例: `5:00` / `10:00` からカウントダウン）。

制限時間が **0:00** になると、盤面の中央付近に **TIME UP** と大きく表示されます。約3秒後に自動で結果画面へ進みます。

## ルール

### 共通ルール

| 項目 | 内容 |
|------|------|
| フィールド | 横 **12** マス × 縦 **24** マス |
| ブロック | 一般的なテトリスと同じ7種類（I, O, T, S, Z, J, L） |
| 出現 | **完全ランダム**（次のブロックは NEXT に表示） |
| ライン消去 | 横一列がすべて埋まると消える（消去前に短い点滅演出あり。詳細は下記） |
| 落下速度 | **時間が経つほど速くなる** |
| ゲームオーバー | 新しいブロックが出せなくなる（積み上がりすぎ） |

#### ライン消去の演出

ラインが揃ってブロックが消える直前、**完成した行が約0.8秒間点滅**します。

| 項目 | 内容 |
|------|------|
| 見た目 | 完成行のブロックが短い間隔で表示／非表示を繰り返す |
| 操作（ラインを揃えたプレイヤー） | 点滅が終わるまで**操作不可**（移動・回転・高速落下） |
| 操作（他のプレイヤー） | **通常どおり操作可能** |
| 点滅後 | ラインが消え、得点やお邪魔ブロックの送信などが反映され、次のブロックが出現する |

### タイムアタックモード

| 項目 | 内容 |
|------|------|
| 制限時間 | **5分**（0:00 になると終了） |
| 時間切れ時 | **TIME UP** を表示したあと、約3秒で結果画面へ |
| 1人用 | 積み上がってゲームオーバーになっても終了 |
| 得点 | 一度に消したライン数に応じて加点 |

| 一度に消したライン数 | 得点 |
|----------------------|------|
| 1列 | 100点 |
| 2列 | 225点 |
| 3列 | 350点 |
| 4列 | 500点 |

- 終了後、**スコアが最も高いプレイヤー**が上位になります。
- 複数人でタイムアタックする場合も、5分経過（TIME UP 表示後）で全員のプレイが終了し、スコアで順位が付きます。

### 対戦モード

| 項目 | 内容 |
|------|------|
| 制限時間 | **10分**（0:00 になると終了） |
| 時間切れ時 | **TIME UP** を表示したあと、約3秒で結果画面へ |

ラインを消すと、**自分以外の全プレイヤー**のフィールドにお邪魔ブロックが送られます。

| 一度に消したライン数 | 送るお邪魔ブロック数（相手1人あたり） |
|----------------------|----------------------------------------|
| 1列 | 2個 |
| 2列 | 4個 |
| 3列 | 6個 |
| 4列 | 8個 |

**お邪魔ブロックの性質**

- フィールドの**下から**せり上がる。
- 出る**列の位置はランダム**。
- ラインの一部としてカウントされるが、**1回のライン消去では消えない**。
- 同じ行がもう一度ライン消去の対象になると、**2回目で初めて消える**（耐久2回）。

**勝敗**

- ゲームオーバーになったプレイヤーは脱落。
- **最後まで残った1人**が勝利（全員ゲームオーバーになった場合は結果画面で順位表示）。
- **10分の制限時間**に達した場合も全員のプレイが終了し、結果画面で順位が付く（生存中のプレイヤーが上位。複数人が残っている場合は消したライン数などで順位）。

## 結果画面

ゲーム終了後（ゲームオーバー、対戦の勝敗確定、または制限時間経過の **TIME UP** 表示後）に表示されます。

| 表示項目 | 内容 |
|----------|------|
| **順位** | 1位〜（モードに応じた並び） |
| **プレイヤー名** | プレイヤー 1 〜 プレイヤー 4 |
| **プレイ時間** | ゲーム開始から終了までの時間 |
| **消したラインの総数** | そのプレイヤーが消したライン数 |
| **スコア** | タイムアタック時のみ |

- **タイトルへ戻る** を押すと、タイトル画面に戻り、もう一度モードを選べます。

### 順位の付け方

| モード | 順位 |
|--------|------|
| タイムアタック | スコアの高い順 |
| 対戦 | 生存したプレイヤーが上位。全員脱落時、または制限時間終了時は消したライン数などで順位 |

## うまく動かないとき（トラブルシューティング）

### タイトルは出るが、モードを選んでも反応しない

| 原因 | 対処 |
|------|------|
| `file://` で `index.html` を開いている | [ゲームの起動方法](#ゲームの起動方法)のとおり、**ローカルサーバー**で `http://localhost:...` を開く |
| JavaScript が読み込めていない | F12 → **コンソール** に赤いエラーがないか確認 |

### コンソールに「MIME type of 'text/plain'」と出る

Windows で `python -m http.server` だけを使っているときに多いです。`.js` ファイルが正しい形式で配信されていません。

**対処:** 次のいずれかに切り替えてください。

```powershell
npx --yes serve .
```

または

```powershell
npx --yes http-server -p 8080
```

### 「このサイトは安全に接続できません」「ERR_SSL_PROTOCOL_ERROR」

`https://localhost` で開いている可能性があります。簡易サーバーは **HTTP のみ** です。

**対処:** アドレスを **`http://localhost:8080`** のように **`http://`** で開く。

### 2人プレイなのにゲーム画面に行かない

2〜4人を選んだあとは、**サブモード（タイムアタック / 対戦）** を選ぶ必要があります。サブモードのボタンがタイトル画面の下に表示されます。

### ゲームパッドが効かない

- USB / Bluetooth で接続されているか確認する。
- ブラウザのタブをクリックした状態で、パッドのボタンを一度押す。
- プレイヤー数と同じ数のパッドがある場合、**コントローラー番号**（0, 1, 2…）がプレイヤー番号と一致しているか確認する。

### favicon.ico の 404 エラー

アイコン用ファイルがないだけです。**ゲームの動作には影響しません。** 無視して問題ありません。

## クイックスタート（最短手順）

1. PowerShell でゲームフォルダに `cd` する。
2. `npx --yes serve .` を実行する。
3. ブラウザで表示された `http://...` を開く。
4. **1人用** をクリックする。
5. **← → ↓** と **スペース** でプレイする。

以上で、テトリスバトルを遊べます。楽しんでください。
